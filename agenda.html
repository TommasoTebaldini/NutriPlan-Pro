🔗 Sincronizzazione Calendario

Il tuo calendario si aggiorna automaticamente su tutti i dispositivi. Scegli il metodo di sincronizzazione che preferisci.

📅 Google Calendar — Sync in tempo reale

Connetti il tuo account Google per sincronizzare automaticamente ogni nuovo appuntamento al momento del salvataggio.

⚠️ Google Client ID non configurato. Clicca il pulsante per le istruzioni su come abilitare questa funzionalità.

🔗 Link di abbonamento — Tutti i dispositivi

Abbonati una volta sola e il calendario si aggiornerà automaticamente ogni ora su Google Calendar, Apple Calendar, Outlook e altri.

Come abbonarcisi:
• Google Calendar: Impostazioni → Aggiungi calendario → Da URL → incolla il link
• Apple Calendar: File → Nuovo abbonamento calendario → incolla il link
• Outlook: Aggiungi calendario → Sottoscrivi da web → incolla il link
⚠️ Accedi per generare il tuo link di abbonamento personale.
📥 Esportazione manuale (alternativa)

Scarica il file .ics per importarlo manualmente nel tuo calendario.

⚙️ Configura Google Client ID

Per abilitare la sincronizzazione con Google Calendar devi configurare un Google OAuth Client ID. Segui questi passaggi:

  1. Vai su console.cloud.google.com
  2. Seleziona il tuo progetto (o creane uno nuovo)
  3. Vai su API e servizi → Credenziali
  4. Clicca Crea credenziali → ID client OAuth 2.0
  5. Tipo applicazione: Applicazione web
  6. In Origini JavaScript autorizzate aggiungi il tuo dominio (es. https://tuodominio.vercel.app)
  7. Copia il Client ID generato
  8. Vai su Vercel → Settings → Environment Variables e aggiungi:
    GOOGLE_CLIENT_ID = <il tuo Client ID>
  9. Esegui un nuovo deploy su Vercel
💡 Suggerimento: Se vedi "Tipo utente: Interno", cambialo in Esterno nella schermata consenso OAuth prima di creare le credenziali.
