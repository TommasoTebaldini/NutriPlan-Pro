⚡ Sincronizzazione AutomaticaAutomatic Sync

Inserisci la tua email e clicca Sincronizza: l'agenda si collega automaticamente al tuo calendario e si aggiorna da sola ad ogni nuovo appuntamento.Enter your email and click Sync: the schedule automatically connects to your calendar and updates itself with every new appointment.

✓ Funziona con Google Calendar · Apple Calendar · Outlook · qualsiasi app calendarioWorks with Google Calendar · Apple Calendar · Outlook · any calendar app

📅 Google Calendar OAuth — Sync immediatoInstant Sync

Autorizza l'accesso diretto a Google Calendar per aggiungere ogni appuntamento in tempo reale al momento del salvataggio.Authorise direct access to Google Calendar to add every appointment in real time when saving.

🔗 URL di abbonamentoSubscription URL

Copia il link e aggiungilo manualmente al tuo calendario.Copy the link and add it manually to your calendar.

• Google Calendar: Impostazioni → Aggiungi calendario → Da URLSettings → Add calendar → From URL
• Apple Calendar: File → Nuovo abbonamento calendarioFile → New Calendar Subscription
• Outlook: Aggiungi calendario → Sottoscrivi da webAdd calendar → Subscribe from web
⚠️ Accedi per generare il tuo link di abbonamento personale.Log in to generate your personal subscription link.

⚙️ Configura Google Client IDConfigure Google Client ID

Per abilitare la sincronizzazione con Google Calendar devi configurare un Google OAuth Client ID. Segui questi passaggi:To enable Google Calendar sync you need to configure a Google OAuth Client ID. Follow these steps:

  1. Vai suGo to console.cloud.google.com
  2. Seleziona il tuo progetto (o creane uno nuovo)Select your project (or create a new one)
  3. Vai su API e servizi → CredenzialiGo to APIs & Services → Credentials
  4. Clicca Crea credenziali → ID client OAuth 2.0Click Create credentials → OAuth 2.0 Client ID
  5. Tipo applicazione: Applicazione webApplication type: Web application
  6. In Origini JavaScript autorizzate aggiungi il tuo dominio (es. https://tuodominio.vercel.app)Under Authorised JavaScript origins add your domain (e.g. https://yourdomain.vercel.app)
  7. Copia il Client ID generatoCopy the generated Client ID
  8. Vai su Vercel → Settings → Environment Variables e aggiungi:Go to Vercel → Settings → Environment Variables and add:
    GOOGLE_CLIENT_ID = <il tuo Client IDyour Client ID>
  9. Esegui un nuovo deploy su VercelTrigger a new deploy on Vercel
💡 SuggerimentoTip: Se vedi "Tipo utente: Interno", cambialo in Esterno nella schermata consenso OAuth prima di creare le credenziali.If you see "User type: Internal", change it to External on the OAuth consent screen before creating credentials.
