🏷️ Diagnosi Nutrizionale — Formato PESNutritional Diagnosis — PES Format

Ogni diagnosi descrive: Problema nutrizionale — Eziologia (correlato a) — Segni/Sintomi (evidenziato da).
Puoi aggiungere più diagnosi PES per lo stesso paziente. Each diagnosis describes: Problem (nutrition diagnosis) — Etiology (related to) — Signs/Symptoms (as evidenced by).
You can add multiple PES diagnoses for the same patient.
